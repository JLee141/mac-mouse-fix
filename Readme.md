# Mac Mouse Fix

This is a fork of [Mac Mouse Fix](https://github.com/noah-nuebling/mac-mouse-fix) by Noah Nuebling.

Mac Mouse Fix, the original product, design direction, and technical foundation were created by Noah. I'm JLee, and this repository is my remix and continuation of that work. New changes and ongoing maintenance in this fork are by me unless noted otherwise.

## Fork status

This fork exists to evolve Mac Mouse Fix in a more experimental, source-first way while staying explicit about where the project came from.

- Original project and core foundation: Noah Nuebling
- Fork-specific updates and ongoing changes in this repo: JLee
- License and upstream notices: preserved

## What's different in this fork

- A license-free remix build for local testing and ongoing development in this repository
- A new `Exceptions` tab for choosing apps that should be excluded from smooth scrolling
- UI and layout cleanup related to the new exceptions flow
- Date-based remix versioning for fork builds
- Ongoing iteration that may diverge from upstream behavior and release timing

## Why use this fork

Use this fork if you want the actively remixed, in-progress version that JLee is customizing and evolving.

Use the original project if you want the official release track, the original documentation, and Noah's upstream distribution and support flow.
